#  House Price Prediction

Regression model to predict California house prices using Linear Regression and Ridge Regression.

##  Project Overview
- **Dataset:** California Housing Dataset (20,640 samples, 8 features)
- **Goal:** Predict house prices based on income, rooms, location etc.
- **Models Used:** Linear Regression, Ridge Regression
- **Best R² Score:** ~0.60

##  Tech Stack
| Category | Tools |
|----------|-------|
| Language | Python |
| Libraries | NumPy, Pandas, Scikit-learn, Matplotlib |
| Environment | Google Colab |

##  Results
| Model | RMSE | R² Score |
|-------|------|----------|
| Linear Regression | ~0.73 | ~0.60 |
| Ridge Regression | ~0.73 | ~0.60 |

## What's in the Notebook
- Data loading and exploration
- Feature analysis and visualization
- Data preprocessing (StandardScaler)
- Model training (Linear & Ridge Regression)
- RMSE and R² evaluation
- Actual vs Predicted plot
- Residual analysis
- Feature importance visualization
